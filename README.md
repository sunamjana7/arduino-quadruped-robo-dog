# 🐶 Arduino-Based Quadruped Robo Dog

An intelligent quadruped robot built using Arduino that mimics real dog-like movements and basic behaviors. The robot is capable of walking, sitting, standing, crawling, and detecting objects using sensors and a camera module.

This project combines **robotics, embedded systems, and mechanical design** using 3D-printed components and servo-based motion control.

---

## 🚀 Features

- 🐾 Quadruped locomotion using servo motors  
- 🪑 Posture control:
  - Sitting  
  - Standing  
  - Crawling  
- 🎯 Object detection using camera module  
- ⚖️ Balance stabilization using MPU6050 (Gyroscope + Accelerometer)  
- 🤖 Real-time control using Arduino  
- 🔧 Modular and extendable design  

---

## 🛠️ Tech Stack

- **Microcontroller:** Arduino Mega  
- **Programming Language:** Embedded C/C++  
- **Motors:** MG995 Servo Motors  
- **Sensor:** MPU6050 (IMU - Gyroscope + Accelerometer)  
- **Camera:** Camera Module (for object detection)  
- **Design:** 3D Printed Body Parts

## ⚙️ Working Principle

The robot uses multiple **MG995 servo motors** to control each leg joint.  
The Arduino Mega sends PWM signals to coordinate movement across all servos.

- Predefined motion sequences control walking, sitting, and crawling  
- MPU6050 sensor helps maintain balance and orientation  
- Camera module enables basic object detection functionality  

---

## ▶️ Setup Instructions

### 1. Software Setup
- Install Arduino IDE  
- Install required libraries:
  - Servo Library  
  - Wire Library (for MPU6050)  
  - MPU6050 Library  

---

### 2. Hardware Setup
- Connect servo motors to PWM pins on Arduino Mega  
- Connect MPU6050 via I2C (SDA, SCL pins)  
- Attach camera module  
- Use external power supply for motors
